# ConvertTool

## 機能要件
元のテキストファイル
00000112 00000115
00000130 00000145

を
1
00:00:01,200->00:00:01,250


2
00:00:01,500->00:00:01,750

とテキストファイルで出力するWindowsアプリケーション。

下2桁は60で割って小数点第4位を四捨五入し、小数点以下の3桁を,に続ける。
間のスペースは「->」に変換し、
行数と改行を2つ加える。
